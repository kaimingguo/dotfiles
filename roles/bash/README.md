# Bash Configuration Files

`BASH` shell configuration files, add `$HOME/.bashrc.d/local` to use private
configuration.

## Sample Custom Configuration

In `$HOME/.bashrc.d/local`, hide macOS Catalina default interactive shell
warning:

```bash
# hide macOS Catalina interactive shell warning:
export BASH_SILENCE_DEPRECATION_WARNING=1
```
